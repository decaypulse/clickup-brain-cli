# ClickUp Brain AI Agent

**Полноценный AI ассистент с сессиями и памятью** — полный аналог Claude Code.

## Возможности

- 🧠 **Сессии и история** — все разговоры сохраняются в SQLite
- 📝 **Память** — можешь продолжить с того места где остановился
- 👻 **Невидимый браузер** — работает в фоне, не мешает
- 🎨 **Markdown рендеринг** — красивый вывод с подсветкой
- 🔄 **Постоянный агент** — не одноразовые вопросы, а полноценный ассистент

## Установка

```bash
# Установить зависимости
pip install playwright rich prompt_toolkit requests

# Установить браузер
python -m playwright install chromium

# Авторизоваться (откроется браузер один раз)
python clickup_capture.py
```

## Использование

### Запуск агента
```bash
# Интерактивный режим
python clickup_agent.py

# Одно сообщение
python clickup_agent.py "Привет!"

# Загрузить сессию
python clickup_agent.py --session a1b2c3d4

# Через батку (Windows)
clickup-agent.bat
clickup-agent.bat "Твой вопрос"
```

### Команды REPL

| Команда | Описание |
|---------|----------|
| `/help`, `/h` | Справка |
| `/new [title]` | Новая сессия |
| `/sessions`, `/ls` | Список сессий |
| `/load <id>` | Загрузить сессию |
| `/history` | История текущей сессии |
| `/clear`, `/c` | Очистить экран |
| `/exit`, `/quit` | Выход |

## Архитектура

```
clickup-cli/
├── clickup_agent.py        # Основной агент
├── clickup_capture.py      # Авторизация (один раз)
├── clickup-agent.bat       # Launcher для Windows
├── browser_profile/        # Сессия браузера (невидимый)
├── sessions.db             # SQLite база сессий и истории
└── .cli_history            # История команд
```

## База данных

Все сессии и сообщения хранятся в `sessions.db`:

```sql
-- Таблица сессий
sessions (id, title, created_at, updated_at)

-- Таблица сообщений
messages (id, session_id, role, content, timestamp)
```

Можешь делать backup:
```bash
cp sessions.db sessions.db.backup
```

## Отличия от clickup_cli.py

| Функция | clickup_cli.py | clickup_agent.py |
|---------|----------------|------------------|
| Сессии | ❌ | ✅ |
| История | Только команды | ✅ Все сообщения |
| Память | ❌ | ✅ SQLite |
| Браузер | Видимый | **Невидимый** |
| Назначение | Быстрые вопросы | **Постоянный агент** |

## Проблемы

### Rate limit
ClickUp ограничивает запросы. Если видишь `Rate limit`, подожди 1-2 минуты.

### Сессия истекла
Если браузер разлогинился:
```bash
python clickup_capture.py
```

### Браузер крашнулся
Агент автоматически восстанавливается. Если не помогает:
```bash
# Удали browser_profile и авторизуйся заново
rm -rf browser_profile
python clickup_capture.py
```
